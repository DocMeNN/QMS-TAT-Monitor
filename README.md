# QMS TAT Monitor

A modern Quality Management System (QMS) Turnaround Time monitoring platform.

## Features

- Real-time TAT tracking
- Interactive dashboard
- SLA breach alerts
- Analytics and reporting
- Modern responsive UI
- API-driven backend

## Tech Stack

### Frontend
- React
- Vite
- TailwindCSS

### Backend
- FastAPI
- PostgreSQL
- SQLAlchemy

### Deployment
- Docker
- Docker Compose
